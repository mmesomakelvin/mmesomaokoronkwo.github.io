# Mmesoma Okoronkwo Portfolio

This repository contains a static portfolio site built on the HTML5 UP `Massively` template and organized into three portfolio tracks:

- Data Analysis
- Data Science
- Data Engineering

## Site structure

- `index.html`: portfolio overview and entry point to the category pages
- `data-analysis.html`: analytics, dashboarding, KPI, and SQL insight projects
- `data-science.html`: Python-led statistical and predictive projects
- `data-engineering.html`: engineering-adjacent work centered on SQL, transformation, and data preparation foundations
- `assets/`: shared CSS, Sass, JavaScript, fonts, and supporting front-end assets
- `images/`: project thumbnails and page imagery

## Working with the site

No build step is required for the current version of the site. Open `index.html` directly in a browser or serve the repository as a static site.

If you want to change the look and feel, the compiled styles live in `assets/css/` and the source Sass files live in `assets/sass/`.
